# Form Info Field

Directus presentation interface for extended information blocks in customer-zone forms.

The field exposes three configuration options:

- `Nazwa pola` (`label`)
- `Ikona` (`icon`)
- `Treść` (`text`)

Use it as a presentation field in a Directus collection. The frontend reads these values through the Directus fields API and can render the configured text as an informational block or modal.

The `Treść` editor includes formatting, list, alignment, link, and HTML source editing tools.
